# v1.0.60

Date: 2026-07-16

- Updated: `wenay-common2` to `^1.0.79`.
- Reason: adopt its bounded dynamic Store memory handling while keeping the
  existing React-facing APIs unchanged.
- Added: Grid Chrome now makes a context-clicked AG Grid RowNode the exclusive
  selection before copy, so right-clicking another row never accumulates an
  unintended multi-selection.
- Added: optional `commandGroups` lets applications label, order and collapse
  growing command groups without a second command or column-state API. Groups are
  keyboard-accessible, the popover scrolls within the viewport, and closed
  one-shot commands leave a short status toast.
- Improved: QA card 47 demonstrates four selectable menu directions, including
  the neutral «Панель» default, and the actual expandable columns/app groups.
- Verification: TypeScript, unit tests, a fresh package build, and the single
  local Vite stand.
