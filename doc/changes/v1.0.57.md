# v1.0.57

Date: 2026-07-15

- Added: `createGridChrome`, an optional adaptive command surface over an existing
  Grid API and `createColumnState`; it supplies grouped column, size, data, and
  declarative table commands without a parallel column-state lifecycle.
- Added: optional `chrome` integration and `Chrome` renderer on `createColumnGrid`.
- Added: compositional row context-menu copy. The clicked row is selected only when
  necessary; app menu items remain intact and domain-row serialization stays injected.
- Added: neutral `.wenayGridChrome*` classes and `--grid-chrome-*` tokens for app themes.
- Fixed: explicitly exported the existing `lib/common/api.js` deep entry, so a clean Vite/esbuild
  install can resolve its unchanged relative CSS imports from the tarball.
