# v1.0.58

Date: 2026-07-15

- Added: `useAiRunClient` and `useFileJobClient`, thin React views over an
  application-owned `wenay-common2` AI-run or file-job client. They observe the
  durable local Store, readiness and (for AI) semantic events without duplicating
  the provider, transport, storage, ACL, raw-input or job lifecycle.
- Updated: `wenay-common2` to `^1.0.78`, adopting its account-filtered Resource
  file/job and provider-neutral AI-run client capabilities at the React boundary.
- Improved: Grid Chrome composes an app's context-menu items after it has selected
  the clicked row, so app actions and injected row copy operate on the same current
  selection.
- Added: live QA-stand cards for Grid Chrome and common2 workflow clients; the
  Grid Chrome trigger now reveals across its whole header hover/focus area while
  retaining a stable reserved slot.
